Add shortcut
============

You could try to add a key mapping (Atom > Open Your Keymap [on Windows: File > Open Your Keymap]) like this one:

    'atom-text-editor':
      'cmd-alt-l': 'editor:auto-indent'
      'alt-cmd-t': 'tree-view:toggle'
